Title and Description:

QR Code Generator 

Lets build a quick response code and stylize it to our needs.
Produces QR codes in many formats.


Installation:
Lets use Python and the library Segno, to build our QR Code. 

You can also output a qr code to the CLI:
    type $ segno 'yourURLlink.com' 
    press enter 
    the output should be generated in the CLI 


Usage:
Can be used to create QR Codes to provide easy access to online information 
through the digital camera on a smartphone or tablet.


Output:
To get your QR code, run the python file to output your QR code img to then be used anywhere.
